# To-Do List Application (Backend Task)

A full-stack To-Do List application built to demonstrate CRUD (Create, Read, Update, Delete) functionality using a Node.js backend and a Vanilla JavaScript frontend.

## 🛠️ Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)
- **Middleware:** CORS, Express.json
